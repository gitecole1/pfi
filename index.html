connaissances en web
html
css
  • STRUCTURE DU HTML
  • Doctype

  • Indique que le contenue suit les standards HTML5.

  • Head

  • C'est le lieux ou l'on retrouve : le titre,et des liens vers des fichiers css.

  • Body

  • Contient le contenu visible de la page, comme les textes et les image.

  • ELEMENTS DU CSS
  • Les Selecteurs

  • Identifie l'élément html à utiliser comme :(p, div, class,hover ...).

  • Block de Déclaration

  • contient une ou plusieurs déclaration entre accolades({}).

  • Déclaration

  • chaque déclaration est composée d'une propriété et d'une valeur, séparées par un deux points':' et terminées par un point virgule ';'.
